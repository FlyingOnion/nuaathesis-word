# nuaa-template-word
南京航空航天大学本科毕设模板word版

南航本科毕设模板LaTeX版链接：点击[这里](https://github.com/nuaatug/nuaathesis)

本模板做了以下改进：
* 将毕业设计中有格式要求的部分都做成了样式，在使用时直接调用样式即可；
* 一级、二级、三级标题均实现自动编号，表号和图号只需执行一个操作也可自动编号（详细说明见docx文档）；
* 目录自动调整
* 页码自动调整
